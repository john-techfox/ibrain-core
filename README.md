## Installation
Add this line to your application's Gemfile:

```ruby
gem 'ibrain-core'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ibrain-core
```

Please install plugin after install this gem
```bash
$ bundle exec rails generate ibrain:install
```
